# Imagens do README

Adicione os arquivos de prévia do projeto nesta pasta para aparecerem no README.

Arquivos esperados:
- `preview-desktop.png`: captura do Dashboard em desktop
- `preview-mobile.png`: captura do Dashboard em celular

Dicas rápidas (Windows):
- Use a Ferramenta de Captura (Snipping Tool)
- Teclas: Windows + Shift + S
- Salve o arquivo como `.png` e coloque aqui em `docs/`

Após adicionar, faça commit e push para que apareça no GitHub.
